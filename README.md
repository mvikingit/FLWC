# FLWC — Financial LLM Wiki Compiler

This repository is the FLWC B0 fixture-only typed schema package scaffold.

## B0 Scope

Allowed in B0:

```text
synthetic fixtures only
typed schema scaffolding
validator scaffolding
unit tests
source-authority archival
Codex fixture-only prompt preparation
```

Not allowed in B0:

```text
real source ingestion
vendor API calls
paid source access
web scraping
model calls
raw LLM output promotion
DuckDB canonical seed
market-data authority cutover
production service
live trading / scanner / order / position sizing
browser profile/session/token migration
```

FLWC compiles financial text knowledge. It does not trade, execute, or decide external admission.
