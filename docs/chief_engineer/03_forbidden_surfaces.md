# FLWC Forbidden Surfaces

Forbidden unless a future named FLWC source-authority node explicitly authorizes the surface:

- real source ingestion
- vendor API calls
- paid source access
- web scraping
- DuckDB canonical seed
- market-data authority cutover
- runtime service creation
- FLWC Console 18110 real service creation
- Node/npm/Vite
- Docker
- production/live trading/scanner/order/position behavior
- broker or execution behavior
- secrets, credentials, browser sessions, browser tokens, SSH private keys
- raw LLM output as truth
- remote mutation without gate
- GitHub remote configuration mutation without explicit gate
- git push/fetch/pull unless explicitly authorized by the active node
- wall-clock substitution for source/event/as-of timestamps
- silent artifact repair while returning ACCEPT

P620 remains a development and research workstation, not production, not live trading, and not market-data authority.
